# Skill: email-cliente

## Regole Voce (da brand-voice.md)

**Cliente/Prospect esterno:**
- Professionale ma diretto
- Numeri e specs prima, spiegazione dopo
- Breve — niente paragrafi lunghi
- Niente: "I hope this finds you well", "Please don't hesitate", filler
- Chiudi con: "Best regards" (EN) o niente (IT informale)

**Team interno:**
- Casual, diretto, "ciao amico mio" se familiare
- Short forms: "pls", "asap", "V"
- Specs/numeri prima, istruzioni dopo

## Procedura

1. Chiedi: **"A chi e' l'email?"** (nome + ruolo/azienda se non specificato)
2. Chiedi: **"Qual e' l'obiettivo?"** (quotazione, follow-up, istruzione interna, altro)
3. Chiedi: **"Hai dati specifici da includere?"** (numeri, date, specs)
4. Genera la bozza
5. Mostra e chiedi approvazione — NON inviare mai autonomamente

## Template per Tipo

### Quotazione a Cliente
```
[Saluto breve]
[Specs spedizione: peso, volume, rotta, servizio]
[Tariffa netta]
[Validita' offerta]
[Eventuale nota su transit time o condizioni]
Best regards
Roy
```

### Follow-up Post-Meeting
```
[Saluto]
[Recap sintetico di quanto discusso — bullet]
[Prossimi passi concordati]
[Richiesta conferma / call to action]
Best regards
Roy
```

### Istruzione Interna a Team
```
[Nome], pls [azione specifica]
[Dettagli: cliente, numero pratica, scadenza]
[Eventuale recap per conferma]
```

## Quality Checklist
- [ ] Nessun filler o frase vuota
- [ ] Numeri/specs presenti se rilevanti
- [ ] Tono corretto per destinatario (esterno vs interno)
- [ ] Breve e actionable
- [ ] Approvazione Roy prima di qualsiasi invio
